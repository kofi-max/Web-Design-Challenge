Summary: Latitude vs. X


The web visualization dashboard seeks to explore an analysis of the effect of temperature, humidity, cloudiness and wind speed against Latitude.Data is sourced from the OpenWeatherMap API for a dataset involving over 500 cities. Mathplotlib, Pandas and Scipy are adopted as dependencies used to accomplished data reading and plotting


The website developed provides an explanation, description and trends through correlations showing each relationship
